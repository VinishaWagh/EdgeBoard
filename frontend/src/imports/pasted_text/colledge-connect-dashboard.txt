  ### 🧩 How ReactBits translates to Figma Features:                                                                                                  
                                                                                                                                                      
  If you are designing this manually in Figma, set up your design components with these Figma equivalents:                                            
                                                                                                                                                      
  1.  Squares  (Grid Background): Create a grid of  50px × 50px  rectangles in a frame. Set opacity to  3%  and randomly color a few cells with       
  #00FFA3  at  10%  opacity to simulate the grid pulse.                                                                                               
  2.  SpotlightCard  (Hover Effects): Create a component variant for your task cards. Add an Ellipse with a radial gradient (Center  #00FFA3  at  15% 
  opacity, outer boundaries transitioning to transparent) and place it behind the text container as a hover highlight.                                
  3.  BorderTrail  (Neon Trails): Create a vector border path on your cards and apply a gradient stroke. Use Figma's Smart Animate prototype          
  transitions on loop to simulate the trail sliding around the card.                                                                                  
  4.  ShinyText  (Header Reflection): Apply a linear gradient fill to the text inside Figma, placing a white/light-grey color stop between two darker 
  color stops. Shift the center color stop in the hover/active variant to create a glare animation.
